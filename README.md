# Islamic Calendar App

A dual calendar app showing Gregorian and Hijri (Islamic lunar) dates side by side.

## Files
- `islamic_calendar.py` — Python backend with date conversion logic
- `index.html` — Interactive calendar widget (HTML/CSS/JS)

## Features
- Gregorian dates in red, Hijri dates in blue on one unified grid
- Islamic events highlighted with amber badges
- Navigate month by month
- No external dependencies
